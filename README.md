# pix-nvim-study

Study neovim function,option,config etc.
Study neovim plugin config.

# use `lazy.vim`

```lua
{
  "pix-nvim-study",
  event = "VeryLazy",
  name = "pix-nvim-study",
  dev = { true },
},
```

# command

- `PRF function_name`: run test functions
- `PRO option_name`: run test option

# Test

```shell
make test
```
